# Moodle-scraper
Scraper hecho para moodle upm, crea un arbol de archivos y descarga los pdfs de las asignaturas 

# Instrucciones de uso
rellena el .env con tus datos y corre el main, esto creara una carpeta moodle en el directorio actual donde se guardará todas las carpetas con 
todos los archivos descargables

renombra el .env.example a .env para que funcione
